# Notion-Clone

Notion-CloneをNode.jsとtypescriptとreactで作成

# DEMO
 

 
# Features

* React
* Reduxを用いた状態管理
* MongoDBによるデータの管理
* Node.js Express
* Typescript
* PostmanによるテストAPI
* react-router-domを使ったルーティング
* Express Validatorのバリデーションチェック
* Dockerによる環境構築
* CI/CDの構築
* JWTを用いた認証
